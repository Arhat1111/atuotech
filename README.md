# Autotech Motors Website Concept — Extraordinary Version

This is a responsive HTML/CSS/JavaScript website concept for Autotech Motors.

## What is included
- `index.html` — complete landing page
- `styles.css` — responsive premium dark/red aesthetic, bento grid, animations
- `script.js` — scroll reveal, parallax, hero slideshow, 3D tilt, configurator, gallery slider

## New unique additions
- Interactive **Cabin Configurator** with product type, vehicle mood, and finish selector
- Animated product scanner / radar preview
- Real Autotech product images from the original website via official asset URLs
- Hero background slideshow using original Autotech banners
- Bento product catalogue layout
- Animated finish wall/gallery
- Fitment Lab timeline animation
- Magnetic CTA buttons, cursor glow, scroll progress, parallax, reveal animations

## How to use
Open `index.html` in a browser. Internet access is required for loading the original Autotech image assets and Google Fonts.
